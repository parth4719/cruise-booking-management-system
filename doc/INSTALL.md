# Installation

## Supported Platforms:

Unix, Cygwin and MacOS.

## Prerequisite

GCC Compiler, GNU make.
If using Cygwin, dos2unix will be required.

```
./install.sh
```

The following will need to be executed:

```
dos2unix install.sh
./install.sh
```

## Build Project:

```
make
```

Excutable binary CBMS file will be placed at `bin` directory.
