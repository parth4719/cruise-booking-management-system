This function updates the available seats column in "Details.csv" each time the booking is confirmed.

Parameters
[in]row_countThe count of no. of rows in the "Detials.csv".
[in]file_idThe row value of the first passenger belonging to particular booking stored in "passenger_details.csv".
[in]cruise_selectedThe number of cruise selected.
[in]cruise_nameThe name of the cruise selected.
[in]no_of_passthe total number of passengers in the specific booking.
[in]total_farethe total fare value of specific booking.
[in]room_bookedThe array contaning the no. of rooms booked and types of rooms booked of specific booking.

After the booking is confirmed this function is called to store the all the booking details into "Bookings_info.csv".

If we fail to open the file, then we need to abort the program, there is nothing to be done.
